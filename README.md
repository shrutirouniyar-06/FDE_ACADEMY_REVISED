# FDE Academy

> **An AI-First Learning Initiative by GlobalLogic**

FDE Academy is a modern, enterprise-grade learning platform designed to showcase the **Forward Deployed Engineer (FDE)** career path, learning programs, mentors, enterprise projects, and AI-driven learning experiences.

Built using **React**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**, the platform provides an engaging and responsive user experience while demonstrating enterprise-level frontend development practices.

---

## 📖 Overview

FDE Academy is a frontend application focused on introducing users to the Forward Deployed Engineer (FDE) ecosystem through interactive storytelling, career pathways, learning programs, and modern web experiences.

The platform highlights:

- AI-first learning philosophy
- Career progression
- Industry-focused programs
- Enterprise projects
- Mentor guidance
- Interactive learning journey
- Modern UI/UX design

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/shrutirouniyar-06/FDE_ACADEMY_REVISED.git
```

### Navigate into the project

```bash
cd FDE_ACADEMY_REVISED
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
